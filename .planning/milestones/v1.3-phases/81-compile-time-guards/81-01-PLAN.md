<objective>
Add compile_error!() guards to all mainnet placeholder pubkey functions across Tax Program, Bonding Curve, and Conversion Vault -- ensuring a mainnet build without real addresses fails at compile time, not silently at runtime with Pubkey::default().

Purpose: Prevent accidental mainnet deployment with placeholder addresses (all-zeros pubkeys) that would brick the protocol.
Output: All three programs have 3-tier feature gating (devnet/localnet/mainnet) with compile_error!() on the mainnet path.
</objective>

Replace the current 2-tier `treasury_pubkey()` (lines 135-145) with 3-tier:

**Conversion Vault constants.rs:**
Replace all three 2-tier functions (crime_mint, fraud_mint, profit_mint) with 3-tier:
- Keep existing `#[cfg(feature = "devnet")]` paths unchanged
- Add `#[cfg(feature = "localnet")]` paths returning `Pubkey::default()`
- Change `#[cfg(not(feature = "devnet"))]` to `#[cfg(not(any(feature = "devnet", feature = "localnet")))]` with compile_error!():

  - fraud_mint: `compile_error!("Set mainnet FRAUD mint address in conversion-vault/src/constants.rs before building for mainnet")`
  - profit_mint: `compile_error!("Set mainnet PROFIT mint address in conversion-vault/src/constants.rs before building for mainnet")`

Add `use std::str::FromStr;` import for `#[cfg(feature = "localnet")]` if needed (Pubkey::default() doesn't need FromStr, so only the devnet cfg needs it -- verify existing import has the right cfg gate).
