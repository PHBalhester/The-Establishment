<objective>
Create BcAdminConfig PDA for the Bonding Curve program and gate all 4 admin instructions behind it.

Purpose: Currently any signer can call admin-only bonding curve instructions (initialize_curve, prepare_transition, withdraw_graduated_sol, close_token_vault). This creates a front-running risk where an attacker could initialize curves with malicious parameters before the deployer. The BcAdminConfig PDA, initialized via ProgramData upgrade authority check, ensures only the verified deployer can perform admin operations.

Output: BcAdminConfig state, initialize_bc_admin instruction, burn_bc_admin instruction, and all 4 admin instructions gated with has_one = authority.
</objective>
