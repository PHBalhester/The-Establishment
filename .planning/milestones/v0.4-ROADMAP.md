# Milestone v0.4: Tax Program

**Status:** ✅ SHIPPED 2026-02-06
**Phases:** 18-21
**Total Plans:** 11

## Overview

Build the Tax Program — the CPI orchestrator that routes swaps through AMM and enforces asymmetric taxation on buys/sells. The Tax Program is stateless (reads tax rates from Epoch Program's EpochState) and acts as the single authorized entry point for all user swaps.

## Phases

**Depends on**: v0.3 Transfer Hook (for hook passthrough), v0.2 AMM (for swap execution)
**Plans**: 5 plans

Plans:
- [x] 18-01-PLAN.md — Program scaffold (constants, errors, events, helpers)
- [x] 18-02-PLAN.md — Tax math functions (TDD: calculate_tax, split_distribution)
- [x] 18-03-PLAN.md — swap_sol_buy instruction (SOL -> CRIME/FRAUD with buy tax)
- [x] 18-04-PLAN.md — swap_sol_sell instruction (CRIME/FRAUD -> SOL with sell tax)
- [x] 18-05-PLAN.md — Integration tests (full CPI chain validation)

- 38 tests passing (27 unit + 11 integration)
