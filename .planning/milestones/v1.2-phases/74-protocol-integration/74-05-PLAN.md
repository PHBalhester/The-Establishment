1. **Token-2022 setup:** Mints must have TransferHook extension pointing to the transfer_hook program. This is critical for all token transfers to work.

2. **Whitelist entries:** All accounts that send/receive Token-2022 tokens must be whitelisted. This includes admin token accounts, curve vaults, and any test user accounts.

3. **Transfer Hook remaining_accounts:** Every token transfer instruction must include the 4-account hook array as remaining_accounts: [extraMeta, wlSource, wlDest, hookProgramId].

4. **Clock manipulation for failure tests:** If the localnet validator supports clock warping, use it to advance past the deadline.
