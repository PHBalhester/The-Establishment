  1. Create `app/components/launch/BuyForm.tsx`:
     Props: `{ curve: CurveStateData, tokenSymbol: "CRIME" | "FRAUD", tokenMint: PublicKey }`

     State: solInput (string for controlled input), submitting (boolean), txStatus (string | null)

     Quote calculation (on input change, debounced ~300ms):
     - Parse solInput to lamports (bigint): `BigInt(Math.floor(parseFloat(solInput) * 1e9))`
     - Guard: if < MIN_PURCHASE_SOL, show "Minimum 0.05 SOL"
     - Call calculateTokensOut(solLamports, curve.tokensSold) from curve-math.ts
     - Call getCurrentPrice(curve.tokensSold) for current price
     - Call getCurrentPrice(curve.tokensSold + tokensOut) for post-trade price
     - Get user's token balance from useTokenBalances() hook (crime or fraud based on tokenSymbol)
     - Cap check: if (currentHoldings + tokensOut) > MAX_TOKENS_PER_WALLET, show warning and cap the display
     - SOL balance check: use useProtocolWallet() solBalance, if solInput > solBalance show "Insufficient SOL"
     - Render PreviewBreakdown with computed values

     Submit handler:
     - Get slippage from useSettings()
     - Calculate minimumTokensOut: `tokensOut * (10000n - BigInt(slippageBps)) / 10000n` (floor division gives floor slippage protection)
     - Get program from getBondingCurveProgram()
     - Build instruction via buildPurchaseInstruction(program, walletPubkey, tokenMint, solLamports, minimumTokensOut)
     - Create Transaction, add instruction, set feePayer and recentBlockhash
     - Sign via useProtocolWallet().signTransaction
     - Send via connection.sendRawTransaction (sign-then-send pattern per MEMORY.md)
     - Confirm via pollTransactionConfirmation from confirm-transaction.ts
     - On success: show success toast/status, clear input
     - On error: parse with parseCurveError from error-map.ts, show error message

     Button: "Buy {tokenSymbol}" -- disabled when: no input, validation errors, submitting, wallet not connected

  2. Create `app/components/launch/SellForm.tsx`:

     - On error: parse with parseCurveError
