    4. Transfer SOL_net to user from sol_vault
    5. Transfer tax to tax_escrow PDA
    6. Decrement tokens_sold by full N (NOT by 85% of N -- tax is on SOL, not tokens)
  - Add `minimum_sol_out` slippage protection parameter (user specifies minimum acceptable SOL_net)
  - Add worked example matching the existing Section 4.4 buy example style
  - Note: sells only valid when curve status is Active (not Filled, not Failed, not Graduated)
