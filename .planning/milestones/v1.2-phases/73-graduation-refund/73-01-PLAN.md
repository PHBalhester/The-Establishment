    **mark_failed.rs** -- Permissionless failure trigger with grace buffer:

    Accounts struct `MarkFailed`:
    - `curve_state: Account<'info, CurveState>` (mut, seeds = [CURVE_SEED, curve_state.token_mint.as_ref()], bump = curve_state.bump). Note: do NOT add a status constraint in the derive -- the handler checks status explicitly so we can return a specific error.
    - No signer constraint (permissionless).

    Handler logic:
    1. Require `curve.status == CurveStatus::Active` -> `CurveError::InvalidStatus`
    2. Get `clock = Clock::get()?`
    3. Compute `failure_eligible_slot = curve.deadline_slot.checked_add(FAILURE_GRACE_SLOTS).ok_or(CurveError::Overflow)?`
    4. Require `clock.slot > failure_eligible_slot` -> `CurveError::DeadlineNotPassed`
    5. Set `curve.status = CurveStatus::Failed`
    6. Emit `CurveFailed { token: curve.token, tokens_sold: curve.tokens_sold, sol_raised: curve.sol_raised, deadline_slot: curve.deadline_slot, current_slot: clock.slot }`

    **prepare_transition.rs** -- Admin-only graduation:

    Accounts struct `PrepareTransition`:
    - `authority: Signer<'info>` (the admin/deployer -- no hardcoded pubkey constraint, matches initialize_curve pattern)
    - `crime_curve_state: Account<'info, CurveState>` (mut, seeds = [CURVE_SEED, crime_curve_state.token_mint.as_ref()], bump = crime_curve_state.bump)
    - `fraud_curve_state: Account<'info, CurveState>` (mut, seeds = [CURVE_SEED, fraud_curve_state.token_mint.as_ref()], bump = fraud_curve_state.bump)

    Handler logic:
    1. Require `crime_curve_state.status == CurveStatus::Filled` -> `CurveError::CRIMECurveNotFilled`
    2. Require `fraud_curve_state.status == CurveStatus::Filled` -> `CurveError::FRAUDCurveNotFilled`
    3. Set `crime_curve_state.status = CurveStatus::Graduated`
    4. Set `fraud_curve_state.status = CurveStatus::Graduated`
    5. Emit `TransitionPrepared { crime_sol_raised: crime_curve_state.sol_raised, fraud_sol_raised: fraud_curve_state.sol_raised, slot: Clock::get()?.slot }`

    **distribute_tax_escrow.rs** -- Escrow -> carnage fund:

    Accounts struct `DistributeTaxEscrow`:
    - `curve_state: Account<'info, CurveState>` (seeds = [CURVE_SEED, curve_state.token_mint.as_ref()], bump = curve_state.bump). Read-only -- no state mutations.
    - `tax_escrow: UncheckedAccount<'info>` (mut, seeds = [TAX_ESCROW_SEED, curve_state.token_mint.as_ref()], bump, constraint = tax_escrow.key() == curve_state.tax_escrow @ CurveError::InvalidStatus). CHECK: SOL-only PDA, validated by seeds + stored key.
    - `carnage_fund: UncheckedAccount<'info>` (mut). CHECK: Carnage SOL vault PDA on epoch program. Validate with constraint: `carnage_fund.key() == Pubkey::find_program_address(&[CARNAGE_SOL_VAULT_SEED], &epoch_program_id()).0 @ CurveError::InvalidStatus`

    Handler logic:
    1. Require `curve_state.status == CurveStatus::Graduated` -> `CurveError::CurveNotGraduated`
    2. Get `rent = Rent::get()?`, `rent_exempt = rent.minimum_balance(0)`
    3. Get `escrow_lamports = tax_escrow.lamports()`, `transferable = escrow_lamports.saturating_sub(rent_exempt)`
    4. Require `transferable > 0` -> `CurveError::EscrowAlreadyDistributed`
    5. Direct lamport manipulation: `**tax_escrow.try_borrow_mut_lamports()? -= transferable`, `**carnage_fund.try_borrow_mut_lamports()? += transferable`
    6. Emit `EscrowDistributed { token: curve_state.token, amount: transferable, destination: carnage_fund.key(), slot: Clock::get()?.slot }`
