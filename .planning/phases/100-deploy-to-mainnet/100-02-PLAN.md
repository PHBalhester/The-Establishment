  <files>.env.mainnet, deployments/mainnet.json</files>
  <action>
    Execute the validated 8-stage deployment pipeline Stages 0 through 4 against mainnet-beta. Follow the exact procedure from Docs/mainnet-deploy-checklist.md.

    **CRITICAL PITFALL AWARENESS:**
    - Source .env.mainnet FIRST: `set -a && source .env.mainnet && set +a`
    - Build WITHOUT --devnet flag (mainnet build)
    - Verify binary address cross-check passes (no devnet addresses in mainnet .so files)
    - Use priority fees for program deploys: `--with-compute-unit-price 1` is fine for pre-deploy (not time-critical)

    **Execution sequence:**

    Stage 0 (Preflight): `./scripts/deploy/stage-0-preflight.sh mainnet`
    - Verify all GO/NO-GO checks pass
    - If any check fails, STOP and report

    Stage 1 (Build): `./scripts/deploy/stage-1-build.sh mainnet`
    - All 7 .so files compiled
    - Hash manifest generated
    - Binary address cross-check passes (no devnet addresses)

    Stage 2 (Deploy 6 Core): `./scripts/deploy/stage-2-deploy.sh mainnet`
    - Deploy AMM, Transfer Hook, Tax, Epoch, Staking, Conversion Vault (NOT bonding curve -- anti-sniper)
    - Each program verified as executable on-chain
    - ~20.79 SOL cost

    Stage 3 (Initialize): `./scripts/deploy/stage-3-initialize.sh mainnet`
    - Runs initialize.ts: creates mints, PDAs, configs, pools
    - Idempotent (safe to re-run if interrupted)
    - Creates deployments/mainnet.json with all addresses
    - ~0.1 SOL cost

    Stage 4 (Infrastructure): `./scripts/deploy/stage-4-infra.sh mainnet`
    - Creates ALT with all protocol addresses
    - Syncs IDLs to frontend
    - Generates constants.ts from deployment.json
    - ~0.01 SOL cost

    After all stages complete, run verify.ts:
    ```bash
    set -a && source .env.mainnet && set +a
    npx tsx scripts/deploy/verify.ts mainnet
    ```

    - CARNAGE_WSOL_PUBKEY from deployments/mainnet.json
    - PDA_MANIFEST from scripts/deploy/pda-manifest.json
