# Deployment Sequence

This document describes the initialization order for the Dr. Fraudsworth system.

## Order Dependencies

The system must be initialized in this specific order due to PDA and whitelist dependencies:

1. **Transfer Hook Program**
   - Initialize WhitelistAuthority PDA
   - Required before: Any whitelist entries can be added

2. **Staking Program**
   - Initialize StakePool (creates StakeVault PDA automatically)

   - Required before: Any stake/unstake operations
