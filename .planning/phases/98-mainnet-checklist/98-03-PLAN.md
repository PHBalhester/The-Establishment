  <name>Task 1: Execute fresh devnet deploy using stage scripts (Stages 0-4)</name>

  <action>
    Execute a fresh devnet deployment following the checklist document step-by-step. This validates CHECK-03 and fixes the devnet state (burned authorities from Phase 97).

    **IMPORTANT:** This task requires sufficient devnet SOL (~30 SOL). If the faucet is rate-limited, request SOL from the user or wait for replenishment. Check balance first with `solana balance keypairs/devnet-wallet.json --url https://api.devnet.solana.com`.

    **Execution order:**

    1. Run `./scripts/deploy/stage-0-preflight.sh devnet` -- verify all preflight checks pass
    2. Run `./scripts/deploy/stage-1-build.sh devnet` -- compile all programs, generate hashes
    3. Run `./scripts/deploy/stage-2-deploy.sh devnet` -- deploy all 7 programs (fresh IDs since authorities were burned)
    4. Run `./scripts/deploy/stage-3-initialize.sh devnet` -- init mints, PDAs, state
    5. Run `./scripts/deploy/stage-4-infra.sh devnet` -- ALT, constants, IDL sync

    **Between each stage:**
    - Follow the GO/NO-GO gate from the checklist document
    - Verify each step's Expected output matches actual output
    - If any step fails or output doesn't match: fix the stage script and/or checklist document, then retry
    - Document any corrections made (these are checklist improvements)

    **After Stage 4:**
    - Run `npx tsx scripts/deploy/verify.ts --cluster devnet` to confirm full deployment health
    - Record any discrepancies between checklist expectations and actual results
    - Patch any corrections back into Docs/mainnet-deploy-checklist.md and/or stage scripts

    NOTE: Stages 5-7 (launch, graduation, governance) are NOT executed in this validation because:
    - Stage 5 (launch curves) requires ~5 SOL per curve to fill
    - Stage 6 (graduation) requires filled curves
    - Stage 7 (governance) requires stable trading
    The validation proves the pre-deploy pipeline (stages 0-4) works. Stages 5-7 were already proven in Phases 94-97 using the same underlying scripts.
  </action>
  <verify>
    Run `npx tsx scripts/deploy/verify.ts --cluster devnet` and confirm all checks pass. Run `solana program show <program_id> --url devnet --keypair keypairs/devnet-wallet.json` for each of the 7 programs and confirm Executable=true.
  </verify>
  <done>
