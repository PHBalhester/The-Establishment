Create a positioned overlay component that displays OHLC + Volume data for a candle:

- Props interface `OhlcLegendProps`: { data: { open: number; high: number; low: number; close: number } | null; volume?: number; prevClose?: number; compact?: boolean }
- When `data` is null, render nothing (return null).

- Styling: Use inline styles or Tailwind classes referencing factory tokens. The legend sits absolutely positioned top-left inside the chart container.

  - Values: text in FACTORY_LEGEND_COLORS.text color (#ecdcc4), font-mono, text-xs.
  - Change%: green if positive, red if negative, using factory-success/factory-error classes.
- Import FACTORY_LEGEND_COLORS from './chart-theme'.
- Use monospace font for values (font-mono class) for the "brass gauge readout" feel from CONTEXT.md.
- Format prices with up to 9 decimal places (SOL memecoin prices can be very small). Use a smart formatter: if price > 1, show 4 decimals; if price > 0.001, show 6 decimals; else show 9 decimals.
- Format volume with toLocaleString() for thousands separators.

         // Get previous candle's close for change%

       }
       // Get volume from volume series

       const latestVol = volData?.[volData.length - 1];

       return;
     }
     const candleData = param.seriesData.get(candleSeriesRef.current!);
     const volData = param.seriesData.get(volumeSeriesRef.current!);
     if (candleData && 'open' in candleData) {
       setLegendData(candleData as { open: number; high: number; low: number; close: number });
       // Find previous candle for change%
       const allData = candleSeriesRef.current?.data();

   let resizeRAF: number | undefined;
   resizeObserver = new ResizeObserver((entries) => {

     resizeRAF = requestAnimationFrame(() => {
       const entry = entries[0];
       if (entry && entry.contentRect.width > 0) {
         chart.applyOptions({ width: entry.contentRect.width });

     });
   });
