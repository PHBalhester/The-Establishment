Create the constants cross-reference matrix with:

**Header:**
```markdown
# Constants Cross-Reference Matrix

**Generated:** [date]
**Source:** 00-concept-inventory.md (CONST-XXX entries)
**Documents:** 12 specification documents

## Normalization Rules
- All percentages normalized to basis points (bps): 1% = 100 bps
- All SOL amounts kept as SOL (not lamports) unless precision requires it
- All time durations kept in their specified unit (slots, minutes, hours)
- Semantic equivalence: '0.01' = '1%' = '100 bps' - NOT a conflict
```

**Matrix format (one section per constant from inventory):**

```markdown
### [Concept Name] (CONST-XXX)

| Document | Value | Location | Context |
|----------|-------|----------|---------|
| Overview | [value] | :[line/section] | "[quote]" |
| Tax_Pool_Logic | [value] | :[section] | "[quote]" |
| [other docs] | - | - | Not mentioned |

**Status:** [AGREEMENT / DISCREPANCY / SINGLE-SOURCE]
**Notes:** [If discrepancy, explain what differs. If single-source, flag for Phase 4]
```

**Priority constants to cross-reference (from 03-RESEARCH.md):**
- LP_FEE_SOL_POOLS (1% / 100 bps)
- LP_FEE_OP4_POOLS (0.5% / 50 bps)

- CARNAGE_TRIGGER_PROBABILITY (1/24)
- YIELD_DISTRIBUTION_SHARE (75%)
- CARNAGE_FUND_SHARE (24%)
- TREASURY_SHARE (1%)
- TOKEN_SUPPLY (for each token)
- POOL_SEEDING_AMOUNTS
- VRF_TIMEOUT_SLOTS

**Search strategy:** For each constant, search ALL 12 documents for:
