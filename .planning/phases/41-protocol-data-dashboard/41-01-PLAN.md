**app/lib/connection.ts** -- Update `getConnection()`:

- Keep the existing function signature (rpcUrl optional parameter).
- The Connection should be a singleton (memoized) since multiple hooks will share it. Create a module-level `let cachedConnection: Connection | null = null` that is reused when the URL hasn't changed. This prevents creating multiple WebSocket connections.

Example connection factory update:
```typescript
let cachedConnection: Connection | null = null;
let cachedUrl: string | null = null;

export function getConnection(rpcUrl?: string): Connection {
  const url = rpcUrl ?? process.env.NEXT_PUBLIC_RPC_URL ?? DEVNET_RPC_URL;
  if (cachedConnection && cachedUrl === url) return cachedConnection;
  cachedConnection = new Connection(url, {
    commitment: "confirmed",
    wsEndpoint: url.replace("https://", "wss://"),
  });
  cachedUrl = url;
  return cachedConnection;
}
```
