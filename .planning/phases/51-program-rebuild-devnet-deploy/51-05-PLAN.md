**Pre-flight: Ensure sufficient devnet SOL**
Check wallet balance on devnet. Need 15+ SOL. If insufficient, request from faucet (may need multiple attempts due to rate limiting).
```bash
source "$HOME/.cargo/env" && export PATH="/Users/mlbob/.local/share/solana/install/active_release/bin:$PATH" && export PATH="/opt/homebrew/bin:$PATH"
solana balance --url devnet --keypair keypairs/devnet-wallet.json
```
If below 15 SOL, request airdrops:
```bash
solana airdrop 5 --url devnet --keypair keypairs/devnet-wallet.json
```

**Step 1: Clean old deployment state**
Delete old mint keypairs and ALT cache to ensure fresh PDAs:
```bash
rm -f scripts/deploy/mint-keypairs/*.json
rm -f scripts/deploy/alt-address.json
```

**Step 2: Deploy all 5 programs**
Use the existing deploy.sh with the Helius devnet RPC (or public devnet if Helius isn't configured). The RPC URL should come from .env file (CLUSTER_URL) or be set explicitly.
```bash
CLUSTER_URL=https://api.devnet.solana.com ./scripts/deploy/deploy.sh
```
Or if Helius is configured in .env:
```bash
./scripts/deploy/deploy.sh
```
This deploys all 5 .so files to devnet using the new keypairs from Plan 04. It also verifies each deployment via `solana program show`.

If deploy fails due to insufficient SOL, airdrop more and retry. If deploy fails due to "account already in use" (old keypair collision -- extremely unlikely with fresh keypairs), the keypair needs regeneration.

**Step 3: Initialize protocol**
Check initialize.ts for LP sizing parameters and adjust to ~2 SOL per SOL pool (CRIME/SOL and FRAUD/SOL) if the defaults are different. Then run:
```bash
npx tsx scripts/deploy/initialize.ts
```
This creates: 3 Token-2022 mints (CRIME, FRAUD, PROFIT), ExtraAccountMetaLists, AdminConfig, 4 AMM pools with seed liquidity, whitelist entries, epoch state, staking pool, carnage fund, WSOL intermediary, and generates pda-manifest.json.

**Step 4: Update shared/constants.ts MINTS**
After initialize.ts generates new mint addresses, update the MINTS object in shared/constants.ts with the new CRIME, FRAUD, PROFIT mint addresses from pda-manifest.json.

**Step 5: Verify deployment**
```bash
npx tsx scripts/deploy/verify.ts
```
This confirms all on-chain accounts exist with correct data.

**Step 6: Verify each program is executable**
```bash
for keypair in keypairs/amm-keypair.json keypairs/tax-program-keypair.json keypairs/transfer-hook-keypair.json keypairs/epoch-program.json keypairs/staking-keypair.json; do
  echo "$(solana-keygen pubkey $keypair):"
  solana program show $(solana-keygen pubkey $keypair) --url devnet
done
```
