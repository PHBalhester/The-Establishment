  <action>
Create `app/components/audio/QuickMuteButton.tsx`:

1. Mark as `'use client'` (uses hooks).

2. Import `useSettings` from `@/hooks/useSettings` and `useAudio` from `@/hooks/useAudio`.

3. Component logic:
   - Read `settings.muted` and `setMuted` from `useSettings()`.
   - Read `isInitialized` from `useAudio()`.
   - If `!isInitialized`, return `null` (don't show before splash screen is dismissed).
   - Derive `isMuted = settings.muted`.
   - Return a `<button>` with:
     - `type="button"`
     - `aria-label="Mute"` (FIXED label per WAI-ARIA APG toggle button pattern -- do NOT change based on state)
     - `aria-pressed={isMuted}` (toggles true/false)
     - `className="quick-mute-btn"`
     - `onClick={() => setMuted(!isMuted)}`
   - Inside the button, render an inline SVG (24x24 viewBox, rendered at 20x20, `aria-hidden="true"`):
     - Speaker body (always visible): `<polygon points="11 5 6 9 2 9 2 15 6 15 11 19 11 5" />`
     - When `isMuted`: Two crossing lines for X (line x1=23 y1=9 x2=17 y2=15, line x1=17 y1=9 x2=23 y2=15)
     - When NOT muted: Two arcs for sound waves (`<path d="M15.54 8.46a5 5 0 0 1 0 7.07" />` and `<path d="M19.07 4.93a10 10 0 0 1 0 14.14" />`)
   - SVG attributes: `fill="none"`, `stroke="currentColor"`, `strokeWidth="2"`, `strokeLinecap="round"`, `strokeLinejoin="round"`

4. Add CSS to `app/app/globals.css` -- append `.quick-mute-btn` styles at the end of the file (NOT inside any @layer, just at document level where other utility/component styles live):

