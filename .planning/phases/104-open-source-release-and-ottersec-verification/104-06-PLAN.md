<objective>
Submit all 6 active mainnet programs for OtterSec verified build badges using the solana-verify CLI. This creates on-chain PDAs linking each program to the public repo + commit hash, then triggers OtterSec's remote builder to independently verify the builds.

Purpose: Verified build badges on Solana Explorer are the gold standard for protocol transparency. They prove the deployed bytecode matches the public source code, giving users cryptographic assurance that what they see on GitHub is what's running on-chain.
Output: All 6 programs submitted for verification, status URLs recorded.
</objective>
