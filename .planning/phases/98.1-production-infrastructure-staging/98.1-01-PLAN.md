Make CSP headers and webhook management cluster-aware so mainnet services use correct URLs.

Purpose: The CSP currently hardcodes devnet Helius URLs. When the mainnet Railway service builds with NEXT_PUBLIC_CLUSTER=mainnet, the CSP must include mainnet.helius-rpc.com (not devnet.helius-rpc.com). Similarly, webhook-manage.ts hardcodes the devnet API base and webhook type -- mainnet needs different values for both.

Output: Two updated files ready for mainnet deployment.
</objective>
