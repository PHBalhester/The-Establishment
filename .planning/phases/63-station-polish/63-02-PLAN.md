### StakingForm.tsx (most complex -- compound Tabs migration)

1. **Add kit imports**: `import { Tabs, TabList, Tab, TabPanel, Card } from '@/components/kit';`

2. **Replace the tab section** (current lines 117-165). The current pattern is:
   ```tsx
   <div className="bg-factory-surface rounded-xl">
     <div className="flex border-b border-factory-border-subtle">
       {TABS.map(tab => (
         <button className="lever-tab flex-1" data-state={...} onClick={...}>{tab.label}</button>
       ))}
     </div>
     <div className="p-4">
       {staking.activeTab === "stake" && <StakeTab ... />}
       {staking.activeTab === "unstake" && <UnstakeTab ... />}
       {staking.activeTab === "claim" && <ClaimTab ... />}
       <StakingStatus ... />
     </div>
   </div>
   ```

   Replace with:
   ```tsx
   <Card>
     <Tabs value={staking.activeTab} onChange={(v) => staking.setActiveTab(v as StakingAction)}>
       <TabList>
         {TABS.map(tab => (
           <Tab key={tab.key} value={tab.key} disabled={isTransacting}>
             {tab.label}
           </Tab>
         ))}
       </TabList>
       <div className="p-4">
         <TabPanel value="stake">
           <StakeTab ... />
         </TabPanel>
         <TabPanel value="unstake">
           <UnstakeTab ... />
         </TabPanel>
         <TabPanel value="claim">
           <ClaimTab ... />
         </TabPanel>
         <StakingStatus ... />
       </div>
     </Tabs>
   </Card>
   ```

   **Type compatibility note**: `staking.setActiveTab` expects `StakingAction` ("stake" | "unstake" | "claim"), but kit Tabs onChange provides `string`. Cast with `(v) => staking.setActiveTab(v as StakingAction)`. This is safe because the only Tab values are "stake", "unstake", "claim".

   **TabPanel conditional rendering**: Kit TabPanel already returns null when inactive (RESEARCH.md confirmed), so the `{staking.activeTab === "X" && ...}` guards are replaced naturally. StakingStatus renders outside TabPanels so it always shows.

3. **Keep all other StakingForm logic unchanged**: useStaking() call, isTransacting derivation, actionDisabled logic, toast notifications, useEffect for status transitions.
