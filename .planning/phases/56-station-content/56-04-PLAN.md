**DocsStation.tsx** (`app/components/station/DocsStation.tsx`):

Create an iframe wrapper for the Nextra documentation site.

1. `'use client'` directive, JSDoc header.
2. DEFAULT EXPORT (required by React.lazy).
3. Read docs URL from environment variable `NEXT_PUBLIC_DOCS_URL` with fallback to `http://localhost:3001` for development:
   ```tsx
   const DOCS_URL = process.env.NEXT_PUBLIC_DOCS_URL || 'http://localhost:3001';
   ```
4. Render an iframe with loading state:
   ```tsx
   const [loaded, setLoaded] = useState(false);

   return (
     <div className="relative" style={{ minHeight: '500px' }}>
       {/* Loading state while iframe loads */}
       {!loaded && (
         <div className="absolute inset-0 flex items-center justify-center">
           <div className="text-center">
             <div className="animate-pulse h-4 w-32 bg-factory-surface rounded mb-2 mx-auto" />
             <p className="text-xs text-factory-text-muted">Loading documentation...</p>
           </div>
         </div>
       )}

       <iframe
         src={DOCS_URL}
         title="Dr. Fraudsworth Documentation"
         className="w-full border-0 rounded"
         style={{ height: '60vh', minHeight: '400px' }}
         sandbox="allow-scripts allow-same-origin allow-popups"
         onLoad={() => setLoaded(true)}
       />
     </div>
   );
   ```
5. The iframe uses `sandbox` attribute with minimal permissions:
   - `allow-scripts`: Nextra needs JS for navigation and search
   - `allow-same-origin`: needed for Nextra's internal routing
   - `allow-popups`: for any external links in docs that open in new tabs
   - No `allow-forms`: docs are read-only
6. If the iframe fails to load (Nextra not running), the loading state persists. Add a timeout fallback:
   ```tsx
   useEffect(() => {
     const timer = setTimeout(() => {
       if (!loaded) setLoaded(true); // Show iframe anyway (may show connection error)
     }, 10000); // 10 second timeout
     return () => clearTimeout(timer);
   }, [loaded]);
   ```
7. Also add a fallback message visible when iframe is shown but may have failed:
   ```tsx
   <p className="text-xs text-factory-text-muted mt-2 text-center">
     Documentation site: <a href={DOCS_URL} target="_blank" rel="noopener noreferrer" className="underline hover:text-factory-text">{DOCS_URL}</a>
   </p>
   ```
