  **A. Carnage WSOL account creation (initialize.ts):**

  Add a new step to `scripts/deploy/initialize.ts` AFTER the existing Carnage vault creation steps (after Step 17). This step creates a WSOL token account owned by the CarnageSigner PDA.

  1. Read the existing file to understand the step numbering and pattern. Steps 15-17 handle Carnage vaults.

  2. Add a new step (e.g., Step 18 or after existing Carnage steps):
     ```typescript
     // Step XX: Create Carnage WSOL account (CarnageSigner PDA-owned)
     // This WSOL account is used by execute_carnage for swap_exempt CPI.
     // Must use explicit Keypair because ATA rejects off-curve (PDA) owners.

     import { createWrappedNativeAccount } from "@solana/spl-token";
     import { Keypair } from "@solana/web3.js";
     import * as fs from "fs";
     import * as path from "path";

     const carnageWsolKeypairPath = path.join(__dirname, "../../keypairs/carnage-wsol.json");

     // Load or generate WSOL keypair (idempotent)
     let carnageWsolKeypair: Keypair;
     if (fs.existsSync(carnageWsolKeypairPath)) {
       const secretKey = JSON.parse(fs.readFileSync(carnageWsolKeypairPath, "utf-8"));
       carnageWsolKeypair = Keypair.fromSecretKey(Uint8Array.from(secretKey));
       logger.info(`Loaded existing Carnage WSOL keypair: ${carnageWsolKeypair.publicKey.toBase58()}`);
     } else {
       carnageWsolKeypair = Keypair.generate();
       fs.writeFileSync(carnageWsolKeypairPath, JSON.stringify(Array.from(carnageWsolKeypair.secretKey)));
       logger.info(`Generated new Carnage WSOL keypair: ${carnageWsolKeypair.publicKey.toBase58()}`);
     }

     // Derive CarnageSigner PDA (owner of the WSOL account)
     const [carnageSignerPda] = PublicKey.findProgramAddressSync(
       [Buffer.from("carnage_signer")],
       epochProgramId
     );

     // Create WSOL account with CarnageSigner as owner
     // Fund with minimum rent-exempt balance (actual SOL will come from sol_vault during swaps)
     const carnageWsolAccount = await createWrappedNativeAccount(
       connection,
       authority,                    // payer
       carnageSignerPda,            // owner (CarnageSigner PDA)
       0,                           // 0 lamports initial (funded per-swap from sol_vault)
       carnageWsolKeypair,          // explicit keypair (ATA rejects off-curve)
       { commitment: "confirmed" },
       TOKEN_PROGRAM_ID             // WSOL uses SPL Token, NOT Token-2022
     );
     ```

     Make this idempotent: check if the account already exists on-chain before creating. Use the existing `accountExists()` helper if available in the initialize script.
