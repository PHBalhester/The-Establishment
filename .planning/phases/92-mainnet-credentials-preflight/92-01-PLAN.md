<task type="checkpoint:human-action" gate="blocking">
  <name>Task 2: Generate mainnet deployer wallet</name>
  <action>
The user must generate the mainnet deployer wallet. Claude cannot do this because the seed phrase
must be written down on paper and stored physically -- this is a security-critical human step.
  </action>
  <instructions>
Run these commands to generate your mainnet deployer wallet:

```bash
source "$HOME/.cargo/env"
export PATH="$HOME/.local/share/solana/install/active_release/bin:$PATH"

mkdir -p ~/mainnet-keys
solana-keygen new --no-bip39-passphrase -o ~/mainnet-keys/deployer.json
```

IMMEDIATELY write down the seed phrase on paper and store it physically separate from your computer.

Then verify the wallet was created:
```bash
solana-keygen pubkey ~/mainnet-keys/deployer.json
```

Record the public key -- you will need it for Phase 98 when funding the wallet.

DO NOT fund this wallet yet. Phase 98 calculates the exact SOL budget first.
  </instructions>
