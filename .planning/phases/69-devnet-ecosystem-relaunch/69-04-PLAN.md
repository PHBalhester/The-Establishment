    - FRAUD -> SOL sell (taxed AMM swap)

    - FRAUD -> PROFIT vault conversion

    - Forward: SOL → CRIME (buy) → PROFIT (vault) → FRAUD (vault) → SOL (sell)
    - Reverse: SOL → FRAUD (buy) → PROFIT (vault) → CRIME (vault) → SOL (sell)

    **If any test fails:** Check the error carefully. Common issues:
    - "account not found" = stale address in shared/constants.ts or pda-manifest.json
    - "InvalidMintPair" = vault program not rebuilt with new mint addresses (re-run build.sh --devnet + redeploy vault)
    - "ConstraintOwner" = program not rebuilt with --devnet feature
    - "insufficient funds" = wallet needs SOL airdrop
