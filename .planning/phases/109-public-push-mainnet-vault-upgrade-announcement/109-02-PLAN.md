    **Step 1: Check deployer wallet balance**

    ```bash
    solana balance 23g7xmrtXA6LSWopQcAUgiptGUArSLEMakBKcY1S59YR --url mainnet-beta
    ```

    **Step 2: Write buffer to mainnet**

    ```bash
    solana program write-buffer target/deploy/conversion_vault.so \
      --url mainnet-beta \
      --keypair ~/mainnet-keys/deployer.json \
      --with-compute-unit-price 50000
    ```

    Record the buffer address from the output.

    **GATE A: Verify buffer exists before proceeding**
    ```bash
    solana program show --buffers --url mainnet-beta --buffer-authority 23g7xmrtXA6LSWopQcAUgiptGUArSLEMakBKcY1S59YR
    ```
    Buffer must appear in the list. If not, STOP -- do not proceed to authority transfer.

    **Step 3: Transfer buffer authority to Squads vault PDA**

    ```bash
    solana program set-buffer-authority <BUFFER_ADDRESS> \
      --new-buffer-authority 4SMcPtixKvjgj3U5N7C4kcnHYcySudLZfFWc523NAvXJ \
      --keypair ~/mainnet-keys/deployer.json \
      --url mainnet-beta
    ```

    **GATE B: Verify authority transfer before creating proposal**
    ```bash
    solana program show --buffers --url mainnet-beta --buffer-authority 4SMcPtixKvjgj3U5N7C4kcnHYcySudLZfFWc523NAvXJ
    ```
    Buffer must appear with vault PDA as authority. If not, STOP -- do not proceed to proposal creation.

    1. Create vault transaction with BPFLoaderUpgradeable::Upgrade instruction
    2. Create proposal

    Record the proposal transaction index.
