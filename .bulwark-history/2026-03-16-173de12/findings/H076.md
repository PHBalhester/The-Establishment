# Finding: H076 - Crank Logs Wallet Balance

## Status: POTENTIAL
## Confidence Score: 5

## Rationale
At `scripts/crank/crank-runner.ts:213-216`, the crank logs wallet balance when it drops below 1 SOL. This appears in Railway logs which are accessible to anyone with Railway project access. However, the crank wallet address is already public (it signs on-chain transactions visible on any Solana explorer), so its balance is publicly queryable by anyone. The log line adds no information that isn't already available on-chain.

## Code Evidence
```typescript
// crank-runner.ts:213-216
if (balance < 1 * LAMPORTS_PER_SOL) {
  console.log(
    `[crank] WARNING: Wallet balance low: ${(balance / LAMPORTS_PER_SOL).toFixed(3)} SOL`
  );
}
```

## Severity Estimate
LOW -- The crank wallet's balance is already public on-chain. Logging it adds no new information disclosure. The only concern is that it confirms which wallet is the crank operator, but this is already determinable from on-chain transaction signing patterns.
