# Finding: H071 - React Override Forces React 19 Globally

## Status: POTENTIAL
## Confidence Score: 6

## Rationale
At `package.json:25-28`, the `overrides` field forces `react` and `react-dom` to version `19.2.3` across all workspaces. This means any dependency expecting React 17/18 APIs will silently receive React 19. React 19 removed several deprecated APIs (e.g., `ReactDOM.render`, `defaultProps` on function components, string refs). If any transitive dependency relies on removed APIs, it will break at runtime. The project uses `@switchboard-xyz/on-demand`, `@coral-xyz/anchor`, WalletConnect libraries, and Privy -- all of which may have React peer dependencies.

## Code Evidence

```json
"overrides": {
  "react": "19.2.3",

}
```

## Severity Estimate
