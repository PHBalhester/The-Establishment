# Finding: H069 - No Minimum Sell Amount in SellForm

## Status: CONFIRMED
## Confidence Score: 7

## Rationale
The SellForm at `app/components/launch/SellForm.tsx:100-158` validates that the parsed amount is >0 (line 114) and checks balance (line 124), but there is no minimum sell amount enforced. Any positive token amount (e.g., 0.000001 tokens = 1 base unit) passes client-side validation. The BuyForm has `MIN_PURCHASE_SOL` (0.05 SOL) but SellForm has no equivalent. The on-chain bonding curve sell instruction would need to be checked for whether it enforces a minimum -- if it does, this is display-only. If it doesn't, dust sells could spam the network with negligible-value transactions.
